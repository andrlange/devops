kappman — Feature-Bereiche

Übersicht aller Bereiche mit ihren Funktionen

⊞ Dashboard
  • Org/Space/App/Service Counters
  • Korifi Health Summary (API, Gateway, kpack)
  • Recent Activity Feed
  • Quick Actions (je nach Rolle)
◎ Organizations
  • Liste aller Orgs mit Space-Count
  • K8s Namespace Mapping anzeigen
  • Admin: Org erstellen/löschen
  • Drill-down → Spaces der Org
▤ Spaces
  • Spaces mit App/Service Counts
  • K8s Namespace Mapping anzeigen
  • Admin: Space erstellen/löschen
  • Drill-down → Apps & Services im Space
⬡ Applications
  • App-Liste mit Status-Badge (STARTED/STOPPED/STAGING)
  • App Detail: Instances, Memory, Disk, Routes
  • Start / Stop / Restart / Restage (HTMX)
  • Scale Instances (HTMX)
  • Environment Variables verwalten
  • Recent Logs anzeigen (HTMX polling)
  • Bound Services anzeigen
  • Admin: App löschen
♦ Services
  • Service Instances im Space
  • Plan, Status, Bindings anzeigen
  • Service Instance erstellen (aus Marketplace)
  • Service Binding erstellen/löschen
  • Service Instance löschen
◈ Marketplace
  • Service Offerings Katalog
  • Plans mit Details (RAM, Storage)
  • Broker Info anzeigen
  • Quick Create → Service Instance erstellen
⧉ Buildpacks
  • Verfügbare Buildpacks (Java, Node, Go, ...)
  • Version & Status
  • ClusterBuilder Status (kpack)
  • Stack Images (Build/Run)
♡ Korifi Status
  • CF API v3 Info & Health
  • Gateway Status (Contour)
  • kpack ClusterBuilder Ready
  • Service Broker Verfügbarkeit
  • Auto-Refresh via HTMX (alle 30s)
⚙ User Management ADMIN ONLY
  • User Liste mit Rollen
  • User erstellen/löschen
  • Rolle zuweisen (Admin/Developer)
  • Org/Space Zuweisungen pro User
  • Passwort zurücksetzen
  • Login: Session-basiert (Spring Security)
